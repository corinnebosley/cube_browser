Writing a Notebook
==================

This tutorial is a guide to using Holoviews and GeoViews to plot and present 
Iris cubes in an IPython or Jupyter notebook.

HoloViews_ is a Python library designed to make plotting and analysing data 
easy and flexible.

.. _HoloViews: http://holoviews.org/index.html

GeoViews_ is a sister Python library designed specifically for the use of 
geographic and oceanographic data.

.. _GeoViews: http://geo.holoviews.org/index.html
